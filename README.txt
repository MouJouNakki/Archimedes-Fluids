Archimedes' Fluids is a mod for Minecraft.

The goal of this mod is to make fluids move more realistically. This mod is currently in early development.

 

Current features:

- Finite fluids

- More realistically flowing fluids

- Pistons and placed blocks push out fluids

- Lavalogging

 

Possible future features:

- Water cycle in the Overworld (Likely)

- Lava cycle in the Nether (Likely)

- Newton's first law of motion (Maybe)

- Splashes (Maybe)

- Fluidlogging any modded fluid (Maybe)
